# Avatar

## Basic usage

<Preview>
  <Story name="Basic usage">
    <>
      <Avatar name="Darth Vader" />
      <Avatar name="Luke Skywalker" />
      <Avatar name="Yoda" />
      <Avatar name="C3PO" color="#DA8E03" />
    </>
  </Story>
</Preview>

If no `color` prop given a color will be generated from the `name` prop.

## With image

<Preview>
  <Story name="With image">
    <>
      <Avatar src="https://upload.wikimedia.org/wikipedia/en/7/76/Darth_Vader.jpg" />
      <Avatar src="https://upload.wikimedia.org/wikipedia/en/9/9b/Luke_Skywalker.png" />

    </>
  </Story>

You can give images or SVG as `src` to `<Avatar>` component.

## With different sizes

<Preview>
  <Story name="Sizes">
    <>
      <Avatar name="Yoda" size="small" />
      <Avatar name="Yoda" size="medium" />
      <Avatar name="Yoda" size="large" />
    </>
  </Story>
</Preview>

You can set `size` prop to `small`, `medium` or `large`. (`medium` by default)

Or you can override CSS variable `--cc-avatar-height`, `--cc-avatar-width` and `--cc-avatar-font-size` through `className` prop.
