## Simple

Button without padding, border and background. The simpliest way to display it. Useful to display an icon button, like modals closing cross.

<Playground>
  <Button simple>Simple button</Button>
</Playground>

## Link

A hyperlink with a button style, just by setting `href` prop.

<Playground>
  <Button href="https://www.google.fr" target="NEW">Link like a button</Button>
</Playground>
